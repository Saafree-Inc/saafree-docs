# Saafree Open Docs

Saafree is an **Enterprise Evolution Operating System (EEOS)** for the AI Agent era — governance by constitutional doctrine and layer laws.

## EEOS

Saafree defines the **EEOS** category: *Enterprise Evolution Operating System*. An EEOS enables organizations to evolve continuously through strategic decision, execution, and adaptation. See **`00-doctrine/eeos-category/`** for the category whitepaper and Saafree architecture.

## What's in this repo

This repository contains the **open** documentation published by Saafree:

- **`00-doctrine/`** — Constitutional doctrine (Articles 0–8), positioning papers, execution doctrine, regulatory materials, and **EEOS category** (whitepaper + architecture).
- **`01-layers/`** — Technical constitution and layer laws (function, logic, physic, security).

The full Saafree OS codebase remains in a private repository.

## Note on links

Some links in these documents point to `docs/02-domains` or `docs/03-infrastructure`. Those paths refer to the full, private codebase and will not resolve in this repo.
